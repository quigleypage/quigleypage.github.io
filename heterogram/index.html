Instructions


You have 6 chances to guess the one-word heterogram.

A heterogram is a word (or phrase) with no repeat letters.

As such, each letter can only be used once per guess.

Guesses do not have to be valid words.

Incorrect letters cannot be reused on subsequent guesses.
